# Channel24 Next.js Clone (Next.js 14 + Tailwind)

## Quick start

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Open http://localhost:3000 (or port shown by Next)

This project is a scaffold for a Channel24-style news site using the Next.js App Router and TailwindCSS.
